# Description

This is little more than a copy of https://github.com/goccmack/gogll/tree/master/examples/json

# How to build

```
# From the Miller base directory
go get github.com/goccmack/gogll/v3
```

```
# Creates lexer/parser \*.go files
gogll json.md
```

```
go build
```

# How to run

```
./goggl-experiment-1
```
